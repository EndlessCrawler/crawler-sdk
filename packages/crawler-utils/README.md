# @avante/crawler-utils

Generic, zero-runtime-dependency utilities for the Endless Crawler SDK. Multi-entry —
import only what you need:

- `@avante/crawler-utils/bi` — the `BigIntish`/`HexString` types + bigint handling
- `@avante/crawler-utils/format` — `formatJSON`, `serialize`/`deserialize`, `formatXML`
- `@avante/crawler-utils/encode` — base64 + data-URI codecs
- `@avante/crawler-utils/seeder` — `makeSeed` (keccak256, `js-sha3` bundled in)
